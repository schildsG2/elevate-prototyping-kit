Elevate Component Library

Visual reference for @g2crowd/elevate. Ported components have live examples. Gray items are planned.

Getting Started

Install: yarn add @g2crowd/elevate

Tailwind preset:

const elevateConfig = require('@g2crowd/elevate/tailwind');

module.exports = {
  presets: [elevateConfig],
  content: [...]
};

Import components:

@import '@g2crowd/elevate/src/css/components.css';
@import '@g2crowd/elevate/src/css/vendor/select2.css';  /* if using select2 */
